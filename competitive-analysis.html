01 — TESIS

La Tesis Central

Por que la orquestacion importa mas que el modelo

"La brecha entre los dos mejores modelos de coding es 0.8 puntos. La brecha entre un buen y mal scaffold de agente es 22 puntos."
— morphllm.com

El Modelo es Commodity

SWE-bench Verified muestra que los top 4 modelos estan separados por menos de 2 puntos (79.6-80.9). Cambiar de Opus a Sonnet cuesta ~1 punto. Cambiar de scaffold cuesta ~22.

El Scaffold es el Diferenciador

WarpGrep v2 + Opus 4.6 logra 57.5 en SWE-bench Pro. Opus 4.5 solo logra 45.89. El mismo modelo sube +11.6 puntos con mejor scaffold. La orquestacion multiplica el talento del modelo.

Poneglyph: Niche Expert

No compite en escala (1 usuario). Compite en sofisticacion: 16 agentes, 24 skills auto-matched, 21 hooks, complexity routing, prompt scoring. Maxima personalizacion para un solo desarrollador.

07 — SWOT

SWOT Actualizado

Fortalezas, debilidades, oportunidades y amenazas a marzo 2026

Fortalezas

  • Unico sistema con decisioning framework completo (prompt scoring + complexity routing + skill matching)
  • Scaffold gap valida approach: +22pts vs model gap 0.8pts
  • 15 specs implementadas en 3 waves (v1.1 → v1.5 → v2.0) — foundation complete
  • Capacidades clave: anti-hallucination, error recovery, graduated autonomy, pattern learning, knowledge graph, skill synthesis, self-optimizing orchestration
  • 87 sesiones de datos reales analizadas con trace analytics
  • 16 agentes especializados + 24 skills auto-matched + 21 hooks de validacion

Debilidades

  • Sin formal verification integration planned Q2 2026
  • Sin spec-driven workflow nativo planned Q2 2026
  • Sin Agent Teams integration external blocker — depende de Anthropic API
  • Task horizon limitado por context window structural — mitigable
  • Sin self-improvement loop automatizado planned Q3 2026

Oportunidades

  • Vericoding (82% en Dafny) — integrar como skill/hook
  • Spec-driven development — crear skill/workflow
  • Self-improving scaffolding (SICA pattern: 17% a 53%)
  • AST-based hallucination detection hook (100% precision)
  • Multi-model cascade (90% cheap, 10% premium = 87% ahorro)

Amenazas

  • METR finding: developers 19% SLOWER con AI tools — perception gap amenaza credibilidad de todo el sector. Validacion empirica es critica.
  • Cursor 2.0 con background agents + git worktree
  • Cline CLI 2.0 con parallel sessions
  • OpenAI Codex 7h autonomous sessions
  • 154% larger PRs with AI (quality concern)
