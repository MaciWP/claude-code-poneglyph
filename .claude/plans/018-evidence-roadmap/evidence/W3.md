## Angle (a) — Degradation vs length: what the curves actually show

- **Lost in the Middle** (Tier A, **2023 era**: GPT-3.5/4, Claude 1.3): U-shaped position effect, >30% drop mid-context; extra context can be net-negative. No 2026 replication found. https://arxiv.org/abs/2307.03172
- **RULER** (Tier A, **2024 era**): GPT-4 effective context 64K of claimed 128K; era heuristic "effective ≈ 25-50% of claimed". No runs on 2026 frontier models found. https://arxiv.org/abs/2404.06654
- **NoLiMa** (Tier A, **early-2025 era**, refuter-confirmed): without lexical overlap, **11/13 models drop below 50% of their own short-context baseline at 32K**; GPT-4o 99.3% (<1K) → 69.7% (32K). Degradation starts as early as 2K-8K. Not re-measured on Claude 4.5+/GPT-5/Fable-5. https://arxiv.org/abs/2502.05167
- **Chroma "Context Rot"** (Tier A/C, **mid-2025 era**: Claude 4, o3, Gemini 2.5, GPT-4.1, 18 models; refuter-confirmed with wording fix): degradation is **gradual, no cliff at any utilization fraction**, occurs even on trivially simple tasks, is stronger with low needle-question similarity and with distractors — i.e., **content shape matters more than volume**. https://www.trychroma.com/research/context-rot
- **2026 frontier (current era, Tier B/C via vendor cards + aggregators)**: near-flat to 128-256K on retrieval-style tasks; the cliff **moved, not disappeared** — MRCR v2 8-needle @1M: Opus 4.6 76% vs Gemini 3 Pro 26.3% (77% @128K → −50pts) vs Sonnet 4.5 18.5%. **No published independent degradation curve exists for Fable-5** — explicit gap.
- **Agentic settings** (the relevant ones): multi-turn −39% (next angle); ReAct append-only context **saturates ~60 interaction rounds then degrades** while context-managed variants improve to 500 rounds (Tier A, **late-2025 era**, arXiv 2512.22087); SWE-Bench Pro: sharp decline as file count grows (Tier A, 2025).
- **Benchmark validity caveat** (Tier A, **critic-corrected** wording): "none of the synthetic tasks achieves an average correlation higher than 0.8" with realistic tasks, RULER averages <0.85 (HELMET, arXiv 2410.02694; the earlier "0.559" point figure was not locatable in abstract/body — dropped) — synthetic curves are weak predictors in BOTH directions.

## Angle (b) — Mitigation policies: measured comparison

**Headline: consolidate-and-restart wins; structured notes second; naive compaction is the measured loser.**

- **Lost in Conversation** (Tier A, **mid-2025 era**: 15 models incl. o3, Claude 3.7, Gemini 2.5; refuter-confirmed exact): multi-turn vs single-turn **−39%** (≈90→65), decomposed −16% aptitude / **+112% unreliability**. Mitigations (GPT-4o, Table 2): FULL 93.0 / **CONCAT 90.9** / RECAP 76.6 / SNOWBALL 65.3 / SHARDED 59.1 — **concatenating everything into a single fresh prompt recovers ~98% of single-turn performance**; recap recovers ~half; temperature 0 does NOT fix unreliability. https://arxiv.org/abs/2505.06120
- **ACON** (Tier A, **2025 era**, GPT-4.1; refuter-confirmed exact): AppWorld no-compression 56.0% ≈ ACON 56.5% at −26% peak tokens, while **FIFO truncation 45.8% and prompt-summarization 43.5%** (LLMLingua 39.3%) — naive compaction costs 10-12.5pts; optimized compression is ~free. https://arxiv.org/abs/2510.00615
- **Factory.ai compression eval** (Tier B `[vendor]`, **2026, GPT-5.2-judge**, refuter-confirmed): probe-based over 36k+ production messages — structured/anchored summaries 3.70 > Anthropic SDK compaction 3.44 > OpenAI compact 3.35 (0-5); **universal failure: artifact tracking (files created/modified) 2.19-2.45/5 for ALL methods** — compaction loses exactly what a coding session needs. https://factory.ai/news/evaluating-compression
- **Memory files vs summarization** (Tier A, **2023 era**, GPT-4; refuter-corrected): MemGPT memory-files **93.4% vs 35.3%** recursive-summarization baseline on Deep Memory Retrieval. (The previously-circulated "full-context baseline 94.4% beats MemGPT" is NOT in the paper — refuted, removed.) https://arxiv.org/abs/2310.08560
- **Workflow/skill memories** (Tier A, **2024 era**): Agent Workflow Memory +24.6%/+51.1% relative (Mind2Web/WebArena); Reflexion HumanEval 91 vs 80 (2023, GPT-4); Voyager skill library (2023). Plain **filesystem tools beat a graph memory system**: 74.0% vs Mem0 68.5% on LoCoMo (Tier B, Letta, 2025, GPT-4o-mini).
- **Subagent compression on downstream CODING quality: no direct measurement exists** (gap, declared). The 90.2% multi-agent result is breadth research at ~15× tokens (seed); Claude Code auto-compact has **no vendor-published task-impact numbers** (Tier C: community failure reports, issue #50888).

## Angle (d) — Code-graph layer (graphify) — gap-closing over `seed-graphify.md`

- **Graph-over-LSP marginal value: unmeasured by anyone** (decisive gap, confirmed closed-as-negative). Best proxies: CodeCompass (Tier B, ~Feb 2026, refuter-confirmed): graph wins ONLY on hidden-dependency tasks (**99.4% vs 76.2% vanilla / 78.2% BM25**, +23.2pp vs vanilla); BM25 ≥ graph on semantic (100% vs 88.9%) AND structural (85.1% vs 76.4%, where graph even underperformed vanilla 79.7%). Tree-sitter KG paper: graph agent = **~90% of explorer quality at ~10× fewer tokens** — a quality dip traded for tokens, not dominance. https://arxiv.org/abs/2602.20048
- **graphify #580 (net-negative tokens)**: closed referencing PR #891; v0.8.27-36 changelogs name no explicit token/caching fix; **no post-fix re-measurement exists** (UNVERIFIED fix efficacy).
- **Aider repo-map** (the 2+ year production precedent): budgeted (1-4k tokens), rank-pruned, ephemeral per-request map — design contrast with graphify's persistent report. **Founding post has zero quantitative ablation** (verified absence); field counter-evidence: weak models get confused by maps, Svelte user runs `--map-tokens 0` after maps "degraded performance and wasted tokens".
- **New independent data since May 2026**: none for graphify; adjacent tool (code-review-graph, Tier B practitioner): avg 8.2× token reduction, range 3.7-49.1× (49× = 27k-file monorepo), **Express single-file changes <1× — net loss**; quality 8.8 vs 7.2/10.

## Counter-evidence (consolidated)

1. Long context did NOT degrade on easy retrieval for 2025-26 models (Gemini 2.5 Pro 91.5→83.1% to 1M `[vendor]`; NIAH essentially solved) — doom narratives from 2023-24 curves overstate.
2. Synthetic benchmarks correlate weakly with real tasks (0.56-0.8, HELMET) — both doom AND all-clear inferences are unreliable.
3. Multi-turn −39% partly reflects the sharding artifact (Tier D critiques); recitation recovers some.
4. Memory systems underperform simple files (Letta 74.0 vs Mem0 68.5); specialized memory < simple files until context no longer fits.
5. Code-graphs: net-negative on small/simple changes (<1×); stale-graph drift; maps can confuse weaker models.

## Verdict

1. **No defensible numeric utilization threshold exists** — measured sources show a gradual, content-dependent gradient (and 2026 models hold to ~128-256K on retrieval). The 40%-rule is unpublished anecdote. The Lead context policy must use **qualitative triggers**: task boundary, observed confusion/repetition, distractor-heavy context, before-a-long-task.
2. **The measured policy ranking for an interactive coding Lead**: (1) consolidate-and-restart from a structured brief (CONCAT ≈98% of single-turn, Tier A); (2) structured notes-to-files for milestones/artifacts (MemGPT 93 vs 35; files beat graph memory; compactors lose artifact state universally); (3) auto-compact as fallback only (naive summarization measurably costs ~10pts).
3. **Agentic context rots by accumulation quality, not length**: −39% multi-turn and ~60-round ReAct saturation are the binding numbers (both 2025 era) — supporting fresh-session-per-task discipline over marathon sessions.
4. **Graphify verdict: WATCHLIST** — graph-over-LSP marginal value is unmeasured anywhere; category wins are confined to hidden-dependency tasks in large repos at a ~10% quality cost; #580's fix is unverified. Optional decisive test: self-run 1-hour A/B on the largest cross-dependency repo (the only way to answer this configuration).
5. **Fable-5-era gap declared**: no independent degradation measurement exists for current Anthropic frontier models — all thresholds inherited from 2024-25 curves are lower-confidence heuristics.
